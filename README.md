# DocFlow // Document Checklist Web Application

ระบบจัดการเช็คลิสต์เอกสารแบบโมเดิร์นที่สวยงาม (Glassmorphism & Sleek Dark Mode) ช่วยให้การเตรียมเอกสารเป็นเรื่องง่าย มีประสิทธิภาพ และข้อมูลไม่สูญหายด้วยระบบบันทึกอัตโนมัติ (LocalStorage)

## 🌟 ฟีเจอร์หลัก
1. **การจัดการ Checklist**: เพิ่ม, ลบ, หรือย้ายเช็คลิสต์เพื่อแยกหมวดหมู่เอกสารยื่นต่างๆ (เช่น วีซ่าท่องเที่ยว, เอกสารภาษี)
2. **การจัดการ Task ย่อย**: เพิ่มรายการเอกสารที่ต้องเตรียม, แก้ไขชื่อเอกสารโดยการดับเบิ้ลคลิก/คลิกที่ชื่อ หรือลบรายการที่ไม่ใช้ออก
3. **สถานะ During / Complete**: สลับสถานะของเอกสารว่าอยู่ระหว่างดำเนินการ (**During**) หรือเตรียมการเสร็จสิ้นแล้ว (**Complete**) เพื่อคำนวณและแสดงแถบความคืบหน้า (Progress Bar)
4. **ระบบค้นหาและฟิลเตอร์**:
   - ค้นหาเช็คลิสต์ที่หน้า Sidebar
   - เลือกฟิลเตอร์เพื่อดูเฉพาะเอกสารที่กำลังทำ (During) หรือเสร็จสิ้นแล้ว (Complete) เท่านั้น
5. **บันทึกอัตโนมัติ**: ข้อมูลทั้งหมดถูกเซฟลงในบราวเซอร์ของคุณ (LocalStorage) เปิดกี่ครั้งข้อมูลก็ยังอยู่ครบ!

## 🚀 วิธีการเปิดใช้งาน

### วิธีที่ 1: เปิดใช้งานโดยตรง (แนะนำสำหรับเริ่มต้นอย่างรวดเร็ว)
1. ดับเบิ้ลคลิกไฟล์ [index.html](file:///C:/Users/satanan/.gemini/antigravity/scratch/document-checklist/index.html) เพื่อเปิดด้วย Web Browser (เช่น Chrome, Edge, Safari, Firefox) ได้ทันที

### วิธีที่ 2: รันผ่าน Local Server (สำหรับนักพัฒนา)
1. เปิด Command Line / Terminal ในตำแหน่งนี้
2. รันคำสั่ง:
   ```bash
   npm run dev
   ```
3. เปิดบราวเซอร์แล้วเข้าไปที่: `http://localhost:3000`
